# ZCBourse
Application web de suivi des bourses mondiales pour débutants. Le projet se concentre sur l'architecture multi-pages, la maîtrise du HTML/CSS et la navigation interne. Une étape structurée vers un outil d'analyse financière complet.

## Plan de travail (Roadmap)
*Ce sur quoi je travaille actuellement pour bâtir les fondations.*

- [ ] Créer la structure HTML de la page d'accueil.
- [ ] Créer les pages HTML pour chaque bourse (6 pages).
- [ ] Relier les pages entre elles (navigation via balises <a>).
- [ ] Appliquer le design global (CSS) pour la mise en forme.

## Objectifs d'apprentissage
- Maîtriser la structure des pages web (HTML).
- Apprendre à créer un parcours utilisateur cohérent (navigation).
- Développer une interface simple et efficace.
